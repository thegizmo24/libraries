1) Merge the /application directory your CodeIgniter /application directory so that all of the files are in their correct locations.
	- config
	- controllers
	- libraries
	- views

2) Update the /application/config/paypal.php with your PayPal API credentials.

3) Refer to the comments at the top of the library for more details on usage.
